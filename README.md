# BukiHTML
BukiHTML, Python ve Tkinter ile yapılmış gelişmiş ve hızlı bir HTML IDE'sidir. BukiHTML ile kendi web sitelerinizi oluşturabilirsiniz.

## Özellikler
- Sözdizimi renklendirme
- Satır numaraları
- Markdown'dan HTML'e dönüştürme
- HTML form menüsü
- Girinti ekleyerek yeni satıra geçme

## Kurulum
BukiHTML'i depodaki "Releases" bölümünden istediğiniz sürümü indirebilirsiniz. Kurmanıza da gerek yok. Eğer isterseniz kaynak kodunu da indirebilirsiniz.
